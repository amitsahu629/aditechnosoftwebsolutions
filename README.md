## ✨ Features

- **Fully responsive** – adapts perfectly to desktops, tablets, and mobile devices.
- **Mobile‑first navigation** – hamburger menu on small screens.
- **SEO optimized** – meta tags, semantic HTML, JSON‑LD schema, and descriptive alt texts.
- **Smooth scrolling** and active section highlighting.
- **Interactive contact form** (client‑side validation with success feedback).
- **Pricing cards** with GST details, renewal info, and “Most Popular” badge.
- **Portfolio section** (“Our Projects”) showcasing recent work.
- **Testimonials** section to build trust.
- **Font Awesome 6** icons for modern visuals.
- **Google Fonts** (Inter) for clean typography.

| Section      | Description                                                                 |

## 🛠 Technology Stack

- **HTML5** – semantic markup
- **CSS3** – custom styles, Flexbox, Grid, media queries
- **JavaScript** – mobile menu toggle, form feedback, active scroll spy
- **Font Awesome 6** (CDN) – icon library
- **Google Fonts** (Inter) – typography
- **Placehold.co** – placeholder images (easily replaceable)

No external frameworks or build steps required – pure static HTML/CSS/JS.

## 🔍 SEO & Performance

- **Meta tags** – description, keywords, author, robots, viewport.
- **Open Graph tags** – improves social media sharing.
- **Canonical URL** – points to live GitHub Pages address.
- **JSON‑LD structured data** (Organization schema) – helps search engines understand business details.
- **Lazy loading** on project images (`loading="lazy"`).
- **Semantic HTML** – `header`, `main`, `section`, `footer`.
- **Descriptive alt attributes** for all images.
- **Fast load times** – minimal external dependencies, lightweight design.
